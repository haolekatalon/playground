# \[WebUI\] Verify Image Present

## Description

Verify if an image is present on page.


:::note

The <code>verifyImagePresent</code> keyword compares images pixel by pixel, and each pixel comparison yields a score from 0.0 to 1.0. The keyword returns <code>true</code> when the average comparison score of all pixels is more than *0.75*.

:::

## Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>to</td><td>TestObject</td><td>Required</td><td>Represent an image.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>

## Returns

<table><tr><th>Param Type</th><th>Description</th></tr><tbody><tr><td>Boolean</td><td><ul><li><p>true if the image if present.</p></li><li><p>false if the image is not present.</p></li></ul></td></tr></tbody></table>

import com.kms.katalon.core.testcase.TestCase as TestCase
import com.kms.katalon.core.testcase.TestCaseFactory as TestCaseFactory
