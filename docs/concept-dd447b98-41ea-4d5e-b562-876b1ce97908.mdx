# Install Katalon Studio on macOS/Windows

This article provides information on how to install Katalon Studio on macOS/Windows.

## Requirements

-   A valid email to register a Katalon Account.
-   An active Internet connection to download Katalon Studio.

## Supported technology


:::note

-   From Katalon Studio version 7.9.1 onwards, we only support 64-bit Windows, macOS, and Linux.


:::

Do a quick check on system requirements before using Katalon Studio. You can refer to this document here: [Supported Environment](/supported-execution-environments/supported-environments-for-katalon-studio-and-katalon-runtime-engine-kre#).

## Download Katalon Studio

To download Katalon Studio, follow these steps:

1.  Sign in to [Katalon TestOps](https://testops.katalon.io/).

2.  On the right side of the navigation bar, click on **Download Studio**.

    <img src="/doc_images/Global_reuse/Common_tasks_warehouse/Katalon_Studio/image/to-download-studio.png" width={700} alt="TestOps download button" />

3.  Select and download the suitable version of Katalon Studio for your system.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_TestOps/to_get_started/kt-download-ks.png" width={500} alt="Download Katalon Studio dialog" />


## Start Katalon Studio in macOS

To start Katalon Studio in macOS, do as follows:

1.  After downloading the .dmg file, double-click on it to proceed with the installation.

2.  Add Katalon Studio to the <code>Application</code> folder when prompted.

    <img src="/doc_images/Documents/Get_Started/Katalon_Studio_Installation/Install_Katalon_Studio_on_macOS_and_Windows/Images/ks-application.png" width={600} />

3.  To start Katalon Studio, double-click on the Katalon Studio application.

    
    :::note

    -   If you are using macOS Catalina, from version 7.9.0 onwards, you no longer have to enable Katalon Studio and Katalon Runtime Engine applications in **System Preferences**. You can start the applications directly as in macOS.

    :::

    Once started, the application should display the splash screen similar to the following screenshot:

    <img src="/doc_images/Documents/Get_Started/Katalon_Studio_Installation/Install_Katalon_Studio_on_macOS_and_Windows/Images/ks-855-activating.png" width={500} />


## Start Katalon Studio in Windows

To start Katalon Studio in Windows, do as follows:

1.  After downloading the .zip file, extract it to the <code>C:\Users\&lt;username&gt;</code> folder.

    
    :::note

    -   For Windows users, if you are extracting Katalon Studio outside of the <code>C:\Users\&lt;username&gt;</code> folder, make sure the current user has the Read/Write permission for Katalon Studio package or runs the software with administrator privileges.

    :::

2.  To start Katalon Studio, double-click on the <code>katalon.exe</code> file.

    -   Ensure you are using the default font size \(set to 100%\) in both Katalon Studio and your current OS to avoid the name field not being displayed on some pop-up windows. To adjust the font size:

        -   For Windows, you can refer to the Microsoft document here: [Edit the font size](https://support.microsoft.com/en-us/windows/change-the-size-of-text-in-windows-10-1d5830c3-eee3-8eaa-836b-abcc37d99b9a).
        -   For Katalon Studio: Go to **Window** &gt; **Preferences** &gt; **General** &gt; **Appearance** &gt; **Colors and Fonts**. Select **Dialog Font** and edit the font size.

## Activate Katalon Studio

To activate Katalon Studio, follow these steps:

1.  The Katalon Studio app launches, then the **Katalon Studio Activation** dialog appears. Sign in to your Katalon account to activate your license. To learn more about license activation steps, see [Activate Katalon Studio license](/administer/katalon-studio-enterprise-and-katalon-runtime-engine-license/activate-a-katalon-license).

    <img src="/doc_images/Documents/Get_Started/Katalon_Studio_Installation/Install_Katalon_Studio_on_macOS_and_Windows/Images/ks-850-se-activation.png" width={500} />

2.  After you are done, click **Activate**.


Now Katalon Studio is ready for use.

## Activate Katalon Studio - Platform Edition

To activate Katalon Studio - Platform Edition, follow these steps:

1.  The Katalon Studio - Platform Edition app launches, then the **Katalon Studio Activation** dialog appears. Sign in to your Katalon account to activate your license. To learn more about license activation steps, see Activate Katalon Studio license.

    <img src="/doc_images/Documents/Get_Started/Katalon_Studio_Installation/Install_Katalon_Studio_on_macOS_and_Windows/Images/ks-850-pe-activation.png" width={500} />

2.  After you are done, click **Activate**.

3.  Select your **Organization** from the dropdown list, then click **OK**.

    <img src="/doc_images/Documents/Get_Started/Katalon_Studio_Installation/Install_Katalon_Studio_on_macOS_and_Windows/Images/ks-850-pe-activation-dialog-full.png" width={500} />


Now Katalon Studio - Platform Edition is ready for use.
