# View your Cloud Studio \(Beta\) test suite report

Learn how to view a report of your Katalon Cloud Studio \(Beta\) test suite.

Before performing this task, you may validate first if your scheduled test run has been completed.

1.  Go to **Executions** &gt; **Test Run List**.

2.  Navigate to the test run, and the date the test run was executed recently under the **Last Run** column: <img src="/doc_images/Documents/Create_tests/Cloud_Studio/Images/test_run_last_run_date.png" width={700} alt="Validate last run of test run." />


You can view the report of your Katalon Cloud Studio \(Beta\) test suite in TestOps. TestOps allows you to centralize and track execution all execution data, including the test results, execution logs, and test report files of your Katalon Cloud Studio \(Beta\) test suite.

1.  Go to **Reports** &gt; **Test Runs**.

2.  Navigate and click your test run. <img src="/doc_images/Documents/Create_tests/Cloud_Studio/Images/view_test_report.png" width={700} alt="Click to view test run report." />


You now have access to your Katalon Cloud Studio \(Beta\) test suite report: <img src="/doc_images/Documents/Create_tests/Cloud_Studio/Images/view_test_execution_report_summary.gif" width={700} alt="View your test run summary report." />

For more information about viewing and analyzing your Katalon Cloud Studio \(Beta\) test suite report, see: [Test runs reports overview](/analyze/reports/view-test-reports/view-test-reports-in-katalon-testops/test-runs-reports-overview)

You can also add Jira defects and link them to your test reports. See: [Link test runs to Jira defects in Katalon TestOps](/analyze/integration-for-test-analyzing/jira-integration/link-test-runs-to-jira-defects-in-katalon-testops)
