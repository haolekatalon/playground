# \[WebUI\] Verify Text Present

## Description

Verify if the given text presents anywhere in the page source.

## Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>text</td><td>String</td><td>Required</td><td>Text to be verified if presenting anywhere in the page source.</td></tr><tr><td>isRegex</td><td>Boolean</td><td>Required</td><td>Indicate whether the expected text is a regular expression.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>

## Returns

<table><tr><th>Param Type</th><th>Description</th></tr><tbody><tr><td>Boolean</td><td>true, if text presents anywhere in the page source; otherwise, false.</td></tr></tbody></table>
