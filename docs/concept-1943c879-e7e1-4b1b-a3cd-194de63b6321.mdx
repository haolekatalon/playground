# View License Details

To view the details of your current license, open Katalon Studio. Go to:

-   macOS: **Katalon Studio &gt; About Katalon Studio**
-   Windows/Linux: **Help &gt; About**

<img src="/doc_images/Documents/Administer/KSE-KRE-Licenses/view-license-details/image/ks-view-license-details.png" width={600} />

In the **About Katalon Studio Enterprise** dialog, you can view the version, edition and build of Katalon Studio, along with the expiration date of your Katalon license.

To learn more about Katalon licenses, see [Types of Licenses](/administer/katalon-studio-enterprise-and-katalon-runtime-engine-license/license-overview).
