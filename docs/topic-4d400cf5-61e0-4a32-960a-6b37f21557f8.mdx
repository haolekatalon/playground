# Katalon Studio installation overview

Katalon Studio can work on macOS, Windows and Linux.

Before the installation, you need to verify whether your computer meets the system requirements to work with Katalon Studio. You can refer to this document for further details: [Supported Environment](/supported-execution-environments/supported-environments-for-katalon-studio-and-katalon-runtime-engine-kre).

To install Katalon Studio, you can refer to the following documents. Instructions in the documents are based on a standard installation using all default settings.

-   [Install Katalon Studio on macOS/Windows](/get-started/katalon-studio-installation/install-katalon-studio-on-macos-windows)
-   [Install Katalon Studio on Linux \(GUI\)](/get-started/katalon-studio-installation/install-katalon-studio-for-linux-gui)
