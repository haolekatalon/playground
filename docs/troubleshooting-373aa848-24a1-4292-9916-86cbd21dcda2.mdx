# Timed out waiting for the driver server to start

Timed out waiting for the driver server to start.

1.  Download the correct Edge driver from the Microsoft website here: [Microsoft Edge Driver](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/)based on your OS build \(Go to **Start** &gt; **Settings** &gt; **System** &gt; **About** and locate the number next to OS Build on the screen\).

2.  Copy downloaded Edge driver and replace it in existing <code>edgedriver</code> folder of Katalon Studio. For example: <code>C:\\Katalon\&#95;Studio\&#95;Windows&#95;64-4.8\\configuration\\resources\\drivers\\edgedriver</code>.
