# Manage test project in Katalon Studio overview

This section shows you how to manage test projects in Katalon Studio:

-   Create a test project: [Create a test project in Katalon Studio](/create-tests/manage-projects/manage-test-projects/create-a-test-project-in-katalon-studio)

    
    :::note

    -   You cannot create a new project in Katalon Studio - Platform Edition.

    :::

-   Open a test project:

    -   In Katalon Studio, see: [Open a test project in Katalon Studio](/create-tests/manage-projects/manage-test-projects/open-a-test-project-in-katalon-studio)

    -   In Katalon Studio - Platform Edition, see: [Open a test project in Katalon Studio - Platform Edition](/create-tests/manage-projects/manage-test-projects/open-a-test-project-in-katalon-studio-platform-edition#)

-   Manage your projects: [Refresh, clean up, and delete a test project in Katalon Studio](/create-tests/manage-projects/manage-test-projects/refresh-clean-up-and-delete-a-test-project-in-katalon-studio)
