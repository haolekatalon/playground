# Katalon TestOps - Common steps

-   Go to **Planning** &gt; **Releases**.

-   Go to **Planning** &gt; **Requirements**.

-   Go to **Test Management** &gt; **Test Cases**.

-   Go to **Test Management** &gt; **Test Suites**.

-   Go to **Test Execution** &gt; **Test Run List**.

-   Go to **Test Execution** and click **Schedule Test Run**.

    The **Schedule Test Run** dialog pops up.

-   Go to **Reports** &gt; **Test Runs**.

-   Go to **Configurations** &gt; **Script Repositories**.

-   Go to **Configurations** &gt; **Test Environments**.

-   Go to **Configurations** &gt; **Agent Setup**.

-   Go to **Configurations** &gt; **Integrations**.

-   Sign in to [Katalon TestOps](https://testops.katalon.io/login) and go to your project.

    The project **Dashboard** appears.

-   Create a new project on [Katalon TestOps](https://testops.katalon.io/):

    1.  Go to TestOps &gt; **Settings** &gt; **Project Management**.

        <img src="/doc_images/Global_reuse/Common_steps/image/to-project-management.png" width={350} alt="project management" />

        The **Project** dashboard appears.

    2.  Click on **Create Project**.

        <img src="/doc_images/Global_reuse/Common_steps/image/to-create-project.png" width={700} alt="project dashboard" />

    3.  Choose your team, then click **Next**.

        <img src="/doc_images/Global_reuse/Common_steps/image/to-choose-team.png" />The **Manage Projects** page appears.<img src="/doc_images/Global_reuse/Common_steps/image/to-manage-project.png" />

    4.  Enter your project name, then click **Create**.

        You have successfully created a new project on Katalon TestOps.

-   Upload test scripts from the Git repository to Katalon TestOps:

    1.  On Katalon TestOps, open your project.

    2.  Go to **Configurations** &gt; **Script Repositories**. Click **Create Git Script Repository**.

        <img src="/doc_images/Global_reuse/Common_steps/image/to-create-git-repository.png" />

    3.  Choose the Git repository source type, then input the Git repository URL, Git username and Git personal access token.<img src="/doc_images/Global_reuse/Common_steps/image/to-script-repository.png" width={700} alt="Script repository" />

    4.  Click **Connect**.

    5.  Choose a Git branch, then input the project name and description.

        <img src="/doc_images/Global_reuse/Common_steps/image/to-choose-branch.png" width={400} alt="choose branch" />

    6.  Click **Create**.

        You have successfully uploaded your test scripts from a Git repository to Katalon TestOps.

-   Log in to TestOps, then click **Settings** at the top right corner.

    From the dropdown menu, select **User Management**. <img src="/doc_images/Global_reuse/Common_steps/image/TO_Access_User_Management.png" alt="TestOps User Management" />

-   Select a test case by clicking on the test case name.<img src="/doc_images/Global_reuse/Common_steps/image/TO-Test_Cases-select_test_case.png" alt="select test case" />

    The Test Case Detail page appears.

-   Select **Edit** to edit the test case details.

    <img src="/doc_images/Global_reuse/Common_steps/image/TO-TestManagement-Edit_test_case.png" alt="Edit test case" />

    The Edit Test Case panel appears.
