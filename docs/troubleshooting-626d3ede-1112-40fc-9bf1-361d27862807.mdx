# Unable to Start Application on this device: Appium directory is invalid

Katalon Studio cannot locate the provided Appium directory.

1.  Double check your Appium directory to make sure it should be as shown below:

    For Windows: \(**Windows** &gt; **Katalon Studio Preferences** &gt; **Mobile** &gt; **Appium Directory**\).

    ```
    C:\Users\<Username>\AppData\Roaming\npm\node_modules\appium
    ```

    For MacOS/Linux: \(**Katalon Studio** &gt; **Preferences** &gt; **Mobile** &gt; **Appium Directory**\).

    ```
    /usr/local/lib/node_modules/appium
    ```
