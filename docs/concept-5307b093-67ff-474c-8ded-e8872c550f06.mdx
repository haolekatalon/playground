# Katalon Platform On-Premises use cases

Katalon Platform On-Premises provides a combination of the abilities to orchestrate software testing cycles along with strict security compliance. In this article, we will explore the various use cases of the Katalon Platform On-Premises and how it can help teams achieve their automation testing goals.

## End-to-end software testing cycle for on-premises environment

Katalon Platform On-Premises \(OP\) delivers a comprehensive testing cycle solution, from test planning, test execution, to test reports and analytics, all in an on-premises environment.

### Test planning

You can create releases and builds for your testing project on Katalon Platform OP, as well as sync up Jira releases:

-   [Create releases and manage your release cycle with Katalon TestOps](/plan/create-and-edit-releases-in-testops).

-   [Link Katalon TestOps releases with Jira releases](/plan/integration-for-test-planning/populate-jira-releases).

-   Within the release, you can [create and manage different builds](/plan/create-and-manage-builds-in-katalon-testops).


### Test creation and organization

You can manage your test scripts by uploading them from your Git repository.

-   [Upload test scripts from Git repository to Katalon TestOps.](/organize/upload-test-scripts-from-a-git-repository/upload-test-scripts-from-the-git-repository-to-testops)

-   [Manage test cases and test suites in Katalon TestOps.](/organize/manage-tests/test-suite/manage-test-suites-in-katalon-testops)


### Test execution

Katalon Platform OP supports executing your test cases by setting up a local test environment.

-   [Create a local test environment with Agent Setup](/execute/cloud-based-test-execution/test-execution-with-testops/local-test-environments/create-a-local-test-environment-with-an-agent).

-   [Make schedule](/execute/schedule-test-execution/schedule-test-runs-in-testops) or [trigger test execution manually](/execute/schedule-test-execution/execute-test-runs-manually-in-testops) in Katalon TestOps.


### Test result reports and analytics

Katalon Platform OP provides reports and analytics that can be leveraged for extracting test performance insights from comprehensive test runs records data.

-   [View test results and execution logs in Katalon TestOps](/analyze/reports/view-test-reports/view-test-reports-in-katalon-testops/view-test-run-results/view-test-results-and-execution-logs-in-katalon-testops/view-test-results-and-execution-logs-in-katalon-testops).

-   [Compare results of multiple test runs in Katalon TestOps](/analyze/analytics/compare-test-results-in-katalon-testops).

-   [Use AI-powered Visual Testing to validate test run results](/analyze/analytics/visual-testing/use-testops-visual-testing).

-   [View test result statistics with TestOps Dashboard](/analyze/reports/view-test-reports/view-test-reports-in-katalon-testops/view-testops-dashboard/testops-dashboard-overview).
