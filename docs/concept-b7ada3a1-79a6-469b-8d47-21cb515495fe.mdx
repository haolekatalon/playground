# Add Custom Fields and Tags to test cases

Besides the test case attributes offered by Katalon, you can create as many as you need to suit your need. Custom Fields and Tags help you do that. You can then organize or query your test cases based on the attributes you have created. To understand more about Custom Field & Tags and how to create them, see [Custom Fields & Tags](/organize/custom-field-and-tags/custom-fields-and-tags).

## Add Custom Fields to a test case

1.  Sign in to [Katalon TestOps](https://testops.katalon.io/) and go to your project.

    The project **Dashboard** appears.

2.  Go to **Test Management** &gt; **Test Cases**.

3.  Select a test case by clicking on the test case name.<img src="/doc_images/Global_reuse/Common_steps/image/TO-Test_Cases-select_test_case.png" alt="select test case" />

    The Test Case Detail page appears.

4.  Select **Edit** to edit the test case details.

    <img src="/doc_images/Global_reuse/Common_steps/image/TO-TestManagement-Edit_test_case.png" alt="Edit test case" />

    The Edit Test Case panel appears.

5.  In the **Custom Field** section, click the **+ Add new** button. <img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Add_Custom_Field_and_Tag_to_Test_Case/images/TO-Test_Management-Add_new_custom_field.png" alt="Add new custom field" />

6.  Select from the dropdown list your custom field key and value. Select **Add** icon to add the custom field. Then click **Save**. <img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Add_Custom_Field_and_Tag_to_Test_Case/images/TO-TestManagement-Add_custom_field_icon.png" alt="Add custom field icon" />


The custom field is added successfully to your test case.<img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Add_Custom_Field_and_Tag_to_Test_Case/images/TO-Test_Management-Custom_Field_added.png" alt="custom field added to test case" />

## Add Tags to a test case

1.  From the Test Case Detail page, navigate to the **Tag** panel.

    <img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Add_Custom_Field_and_Tag_to_Test_Case/images/TO-Test_Management-_Tags_panel.png" alt="Tags panel" />

2.  To create a new tag, enter a tag into the **Select tags** field and click **\(new tag\)** from the dropdown list.<img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Add_Custom_Field_and_Tag_to_Test_Case/images/TO-Test_Management-add_new_tag.png" alt="Add new tag" />

    A new tag is successfully created and added to your test case.<img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Add_Custom_Field_and_Tag_to_Test_Case/images/TO-Test_Management-new_tag_added.png" alt="Tag added to the test case" />

3.  To select among the tags you have created beforehand, select the dropdown list icon at the end of **Select tags** field.

    The dropdown list containing your pre-existing tags appear.<img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Add_Custom_Field_and_Tag_to_Test_Case/images/TO-_Test_Management_-_Select_tags_dropdown.png" alt="Select tags dropdown list icon" />

4.  Click a tag within the dropdown list to add it to your test case.

5.  To remove a tag from your test case, click the remove tag icon and your tag will be removed from the test case.<img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Add_Custom_Field_and_Tag_to_Test_Case/images/TO-Test_Management-remove_tags.png" alt="remove tags" />
