# Network security errors

For users who are in a highly secure environment, you might encounter a situation where you fail to execute test scripts, integrate Katalon Studio, and/or access Katalon TestOps or TestCloud.

This is caused by your network access configurations.

1.  To solve this issue, you need to contact your IT team to whitelist the following domains:
