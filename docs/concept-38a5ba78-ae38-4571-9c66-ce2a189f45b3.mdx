# Combine Multiple Data Sources in Katalon Studio

This article shows you how to execute your automation test using predefined test data from multiple sources.

## Requirements

-   You need an active Katalon Studio Enterprise license. To learn more about activating licenses, you can refer to this document: Activate Katalon license.


## Expand the data binding section in Katalon Studio

To expand the **Data Binding** section, go to your test suite, click **Show Data Binding**. This feature help you bind the predefined test data files and manage variable binding with the Test Suite you want to run.

<img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/ks-841-show-data-binding.png" />

In the extended **Data Binding** section, there are two tables:

-   **Test Data**: Specify here the data files for your test execution.
-   **Variable Binding**: This displays all variables of the selected test case. See also [Test case Variables](/create-tests/data-driven-testing/test-case-variables#).

## Test data table in Katalon Studio

### Adding data sources in Katalon Studio

1.  In the **Test Data** table, click **Add** to add data file\(s\). A **Test Data Browser** dialog opens.

    <img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/ks-841-add-test-data.png" width={700} />

2.  Select a combination of data files you wish to use for variable binding in the **Test Data Browser** dialog. Click **OK**.

    <img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/ks-841-test-data-browser.png" width={500} />

    The selected test data files appear in the **Test Data** table.

    <img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/ks-841-test-data.png" width={650} />


### Modify data range in Katalon Studio

To specify the data range, double-click on the cell under the **Data Iteration** column of each data files.

<img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/ks-841-data-iteration.png" width={500} />

<table><tr><th>Option</th><th>Description</th></tr><tbody><tr><td>Run all rows</td><td>To use all the data rows in the data file in the test execution.</td></tr><tr><td>Run from row ... to row ...</td><td>To use the data range from a particular row to another particular row in the data file during the test execution.</td></tr><tr><td>Run with specific rows</td><td><p>To use the specific data rows in the data file during test execution. You can use <strong>comma</strong> and <strong>hyphen</strong> characters to define the rows.</p> <p>For example:</p> <ul><li>to use three data rows (row 1, row 2, row 3), enter: 1,2,3</li><li>to use six data rows (row 1, row 2, row 3, row 4, row 5, row 9), enter: 1-5,9</li></ul></td></tr></tbody></table>

## Variable binding table in Katalon Studio

After adding the test case into the test suite, Katalon Studio automatically imports all variables of the selected test case into the **Variable Binding** table.

1.  Katalon Studio allows you to **Set Type** for variables all at once if the variables have the same type. In this case, **Username** and **Password** have the same type as **Data Column**. Highlight both rows, click **Set Type &gt; Data Column**.

    <img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/KS-830-Set-data-type.png" width={500} alt="Set Data Type" />

2.  In the **Test Data** column, click on each row to choose the test data files for execution.

    <img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/KS-830-Choose-test-data.png" width={500} alt="Choose Test data for Data Type" />

3.  In the **Value** column, click on each row to specify the data field in the selected data file.

    <img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/KS-830-Add-variables-value.png" width={500} alt="Choose Value for Data Type" />

4.  Save the test suite when you finish.

    <img src="/doc_images/Documents/Create_tests/Data_Driven_Testing/combine-multiple-data-sources/Pictures/KS-830-Final-result.png" width={600} alt="Final result" />

    During execution, the *username* variable looks for the *Username* column of the *valid-accounts* excel file while the *password* variable searches for the *Password* column of the *valid-accounts* CSV file.

2.  Specify the data used in the **Value** cell.
