# Configure Linux for Web Service testing in Katalon Studio

This article provides some additional setup to perform Web Services testing in Katalon Studio for Linux.

To see the Web Service response section and email template content, do as follows:

1.  Install <code>libwebkitgtk-3.0-0</code>.

    In the Terminal, type this:

    ```
    apt-get install libwebkitgtk-3.0-0
    ```


You can now execute API testing with Katalon Studio for Linux.
