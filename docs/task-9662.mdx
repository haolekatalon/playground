# KatOne Admin - Common steps

-   Go to **Settings** &gt; **Organization Management**.

-   Go to **Settings** &gt; **License Management**.

-   Go to **Settings** &gt; **User Management**.

-   Go to **Settings** &gt; **Team Management**.

-   Go to **Settings** &gt; **Project Management**.

-   Go to **Settings** &gt; **Product Utilization**.

    The **License Utilization** page appears.

-   Go to **Settings** &gt; **TestOps Homepage**.
