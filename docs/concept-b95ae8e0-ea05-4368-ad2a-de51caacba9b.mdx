# \[WebUI\] Verify Element Text

## Description

Verify text of an element.

Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>to</td><td>TestObject</td><td>Required</td><td>Represent a web element.</td></tr><tr><td>expectedText</td><td>String</td><td>Required</td><td>Text of the element to verify.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop</td></tr></tbody></table>

## Returns

<table><tr><th>Param Type</th><th>Description</th></tr><tbody><tr><td>boolean</td><td><p><strong>true</strong> if the element has the desired text, otherwise <strong>false</strong>.</p></td></tr></tbody></table>
