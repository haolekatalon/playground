# \[WebUI\] Click

## Description

Click on the given element. From Katalon Studio version 8.2.0 onwards, if the target element is behind a loading overlay, Katalon repeatedly tries clicking the element for a period configured in **Project &gt; Settings &gt; Execution &gt; Default wait for element timeout**.

<img src="https://github.com/katalon-studio/docs-images/raw/master/katalon-studio/docs/wait-for-element-timeout/KS-OVERLAY-Default-timeout-settings.png" alt="Default wait for element timeout settings" />

## Example

In this example, we want to click on the **Make Appointment** button. By default, the **Default wait for element timeout** setting is for 30 seconds. If the **Make Appointment** button is behind a loading overlay, Katalon will try clicking the button for 30 seconds maximum.
