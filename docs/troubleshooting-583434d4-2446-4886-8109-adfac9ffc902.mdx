# Katalon Mobile Recorder and SetText keyword cannot perform on an EditText element of the Flutter-based application

1.  Create a [SetText Custom keyword](/create-tests/keywords/custom-keywords/flutter-based-application-testing-with-custom-settext-keyword-in-katalon-studio).

2.  Run the test again in script mode.
