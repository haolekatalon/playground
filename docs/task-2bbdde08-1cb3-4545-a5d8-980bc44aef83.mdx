# Schedule a Cloud Studio \(Beta\) test suite

Learn how to schedule a test suite in Katalon Cloud Studio \(Beta\).

To schedule a test suite in Katalon Cloud Studio \(Beta\), fulfill the following requirements:

-   An active Katalon TestCloud subscription or trial. See: [TestCloud Trial](/administer/katalon-platform-packages/testcloud-feature-comparison#).

-   A Cloud Studio \(Beta\) test suite. See: [Create a Katalon Cloud Studio \(Beta\) test suite](/create-tests/create-tests-using-katalon-cloud-studio/create-and-manage-a-test-suite/create-a-katalon-cloud-studio-beta-test-suite).


To schedule an execution, follow these steps:

1.  Sign in to [Katalon TestOps](https://testops.katalon.io/login) and go to your project.

2.  Go to **Executions** and click **Schedule Test Run**.

3.  On the Schedule Test Run modal window, provide the following details as shown in the demo below: <img src="/doc_images/Documents/Create_tests/Cloud_Studio/Images/Katalon_Cloud_Studio_schedule_run_test_suite.gif" width={700} alt="Schedule run of a Katalon Cloud Studio test suite." />

    -   Your test run name
    -   The script repository that stores your project

    -   Your Katalon Cloud Studio \(Beta\) test suite

    -   Your preferred environment. See: [Supported environments](/create-tests/create-tests-using-katalon-cloud-studio/execute-cloud-studio-beta-tests/execute-katalon-cloud-studio-beta-tests#).

    -   Your preferred schedule \(**Schedule At**\).

        -   If you wish to repeat your test run, click the slider button to turn on the **Repeat** test run option and indicate when \(**Start**\) and how often \(**Every x weeks**\) the test run is scheduled to be executed.

4.  Click **Schedule** to schedule your test run, or click the dropdown then select **Run** to schedule your test run with no repeated execution.


The test execution is then added to **Test Run List**.<img src="/doc_images/Documents/Create_tests/Cloud_Studio/Images/testops_cloud_studio_test_run_list.png" />
