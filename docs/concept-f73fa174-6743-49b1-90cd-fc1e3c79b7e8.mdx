# \[WebUI\] Select Option By Value

## Description

Select the option with the given value.

<img src="/doc_images/Documents/Create_tests/Keywords/Keywords_Description_In_Katalon_Studio/Web-UI-Keywords/webui-select-option-by-value/ks-webui-select-by-value.png" width={700} />

## Parameters

<table><tr><th>Parameter</th><th>Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>to</td><td>TestObject</td><td>Required</td><td>Represent a web element</td></tr><tr><td>value</td><td>String</td><td>Required</td><td>Value of the options to be selected. For example:<ul><li>Tokyo CURA Healthcare Center</li><li>Hongkong CURA Healthcare Center</li><li>Seoul CURA Healthcare Center</li></ul></td></tr><tr><td>isRegex</td><td>Boolean</td><td>Required</td><td><strong>true</strong> if the value is a regular expression, otherwise <strong>false</strong></td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop</td></tr></tbody></table>

```
import com.kms.katalon.core.testcase.TestCaseFactory as TestCaseFactory

'Fill in login information'

WebUI.closeBrowser()

```

## Example \#2

You may also select the option that contains "Hongkong" with a regular expression. You may make the following changes:

```
'Select the option that contains \"Hongkong"\'
WebUI.selectOptionByValue(findTestObject('Page_CuraAppointment/lst_Facility'), 'Hongkong.*', true)
```

<code>Hongkong.&lt;</code> is the sample regular expression. This matches all the strings that contain "Hongkong", and is followed by zero or more of any character.
