# Unable to start application

This happens when:

-   Missing one or more dependencies.
-   Outdated or incompatible dependencies.
-   Wrong dependencies installation order.
-   Broken WebDriverAgent build package.

1.  Reinstall dependencies.

2.  Rebuild WebDriverAgent. Learn how to rebuild WebDriverAgent at [Install WebDriverAgent](/create-tests/manage-projects/set-up-projects/mobile-testing/ios/mobile-install-webdriveragent-for-real-ios-devices-in-katalon-studio).
