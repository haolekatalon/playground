# \[WebUI\] Authenticate

## Description

Navigate to a page that requires authentication. System will enter username and password.

## Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>url</td><td>String</td><td>Required</td><td>URL of the page to navigate.</td></tr><tr><td>userName</td><td>String</td><td>Required</td><td>Username to authenticate.</td></tr><tr><td>password</td><td>String</td><td>Required</td><td>Password to authenticate.</td></tr><tr><td>timeout</td><td>int</td><td>Required</td><td>Time to wait since navigating to the page until entering username.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>
