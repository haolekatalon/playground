# Product introductions

Name the ID in the following format: &lt;product name&gt;-introduction. For example: ks-introduction

**1. Katalon Studio**

[Katalon Studio](https://docs.katalon.com/katalon-studio/docs/katalon-studio-overview.html) is a productive IDE for test case generation, with cross-platform execution.

**2. Katalon TestOps**

[Katalon TestOps](https://docs.katalon.com/katalon-analytics/docs/overview.html) is an enterprise-ready platform for QA Orchestration, test analytics, and reporting.

**3. Katalon Recorder**

[Katalon Recorder](https://docs.katalon.com/katalon-recorder/docs/overview.html) is a free, Selenium IDE alternative, lightweight web extension for automating actions and automated testing on the browser. Currently, we are available on Chrome, Firefox and Edge.

**4. TestCloud**

Katalon TestCloud is a cloud-based test execution environment equipped with the most common browsers and operating systems.

**5. Katalon Studio Enterprise**

[Katalon Studio Enterprise](https://docs.katalon.com/katalon-studio/docs/katalon-studio-overview.html) is an advanced, robust, and scalable solution for businesses of any size.

**6. Katalon Runtime Engine**

[Katalon Runtime Engine](concept-327a24d1-cba1-4b0b-9337-409f7a3d1abc.mdx) is the test execution add-on of Katalon Studio for executing and scheduling tests in CI/CD pipeline or via the command-line interface.

**7. Katalium Framework**

[Katalium Framework](https://docs.katalon.com/katalium-framework/docs/katalium-framework-overview.html) is a framework that provides a blueprint for test automation projects based on TestNG and Selenium.
