# Fail to start Appium server in 30 seconds

Katalon Studio can't start Appium server within 30 seconds \(default timeout\).

1.  You can increase this timeout value from this settings: **Project** &gt; **Settings** &gt; **Execution** &gt; **default** &gt; **Default wait for element timeout \(in seconds\)**.
