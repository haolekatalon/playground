# Detect abnormal Web Services in Katalon TestOps

-   You need to subscribe to the Ultimate plan. To request a trial, see [TestOps Trial Plans](/administer/katalon-platform-packages/katalon-platform-trial-plans).


Katalon TestOps automatically detects abnormal Web Services. This feature enhances failure detection.

Follow these steps:

1.  Sign in to [Katalon TestOps](https://testops.katalon.io/login) and go to your project.

2.  Go to **Test Management** &gt; **Web Services**, then select the **Anomalies** tab.

    You can view a list of API requests which have taken less or more time than usual to respond here.

3.  To export an API request as HAR, go to the detail page of each Web Service call.
