# Execute Katalon Cloud Studio \(Beta\) tests

You can only schedule a test run with a repository that has both published Katalon Cloud Studio \(Beta\) and Katalon Studio \(KS\) test cases with TestCloud. These test cases must be added to a test suite.

## Supported environments

For executing Cloud Studio \(Beta\) test suites, TestCloud supports the following browsers and operating systems.

<table><tr><th>Browser</th><th>Version on Linux</th></tr><tbody><tr><td>Chrome</td><td>113</td></tr><tr><td>Firefox</td><td>113</td></tr></tbody></table>
