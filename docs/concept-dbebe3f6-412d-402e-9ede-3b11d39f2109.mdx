# \[WS\] Send Request

## Description

Send an HTTP Request to a web server.

Keyword name: <code>WS.verifyElementText</code>

## Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>request</td><td>RequestObject</td><td>Required</td><td>The object that represents an HTTP response.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>

## Returns

<table><tr><th>Param Type</th><th>Description</th></tr><tbody><tr><td>ResponseObject</td><td>A response object</td></tr></tbody></table>
