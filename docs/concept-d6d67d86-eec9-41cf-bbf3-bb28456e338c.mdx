# \[WebUI\] Click Offset

## Description

Click on the given element with the relative position \(x, y\) from the in-view center point of that element. Starting **Katalon Studio** version 8.2.0, if the target element is behind a loading overlay, **Katalon Studio** repeatedly tries to click the element for a period configured in **Project &gt; Settings &gt; Execution &gt; Default wait for element timeout**.

<img src="https://github.com/katalon-studio/docs-images/raw/master/katalon-studio/docs/wait-for-element-timeout/KS-OVERLAY-Default-timeout-settings.png" alt="Default wait for element timeout settings" />

A negative value means an offset above the point.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify failure handling schema to determine whether the execution should be allowed to continue or stop.
