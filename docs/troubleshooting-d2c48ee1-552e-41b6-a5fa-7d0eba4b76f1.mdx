# Unable to record on Internet Explorer

Unable to record on Internet Explorer.

1.  Open Internet Explorer, select the **Tools** button, and then select **Manage add-ons**.

2.  Under **Show**, select **All add-ons**.

3.  Select the **RecorderExtension.RecorderBHO** add-on, **Enable,** and then select **Close**.
