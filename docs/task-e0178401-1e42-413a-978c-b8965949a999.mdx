# Download report files using REST API

1.  After selecting a test run, go to **Test Results** tab and select the **Copy as REST API** icon.

    <img src="/doc_images/Documents/Analyze/Reports/View_Test_Reports/View_reports_in_TestOps/View_test_run_results/View_test_run_results_and_execution_logs/Download_test_report_file/Image/TO-Test_Results-copy_as_REST_API.jpeg" alt="Copy as REST API" />

2.  Navigate to **Files** tab, hover on your desired report file, copy the file ID, which will appear at the lower left corner of the page.

    <img src="/doc_images/Documents/Analyze/Reports/View_Test_Reports/View_reports_in_TestOps/View_test_run_results/View_test_run_results_and_execution_logs/Download_test_report_file/Image/TO-Test_Runs-Files_ID.jpeg" alt="Test reports file ID" />

3.  Go to Katalon Swagger UI and paste the file ID to the required field. Select **Download file**.

    <img src="/doc_images/Documents/Analyze/Reports/View_Test_Reports/View_reports_in_TestOps/View_test_run_results/View_test_run_results_and_execution_logs/Download_test_report_file/Image/Swagger_UI-_paste_file_ID.jpeg" alt="Paste file ID to Swagger UI" /><img src="/doc_images/Documents/Analyze/Reports/View_Test_Reports/View_reports_in_TestOps/View_test_run_results/View_test_run_results_and_execution_logs/Download_test_report_file/Image/Swagger_UI_-_download_file.jpeg" alt="Select Download file on Swagger UI" />
