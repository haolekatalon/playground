# Perform Touch ID with iOS device simulator

-   You have installed Xcode

1.  Here is our [sample project](https://drive.google.com/file/d/11uclypU3ZeHTGtUNJUVCFJvuSKHU2hkA/view?usp=sharing). Select the kind of simulator that you want to build the .app file.

    <img src="/doc_images/Documents/Create_tests/Manage_Projects/Set_Up_Projects/Mobile-Testing/iOS/Perform_Touch_ID_with_iOS_device_simulator/Xcode-_select_the_simulator_to_build_.app_file.jpeg" alt="select the simulator to build the .app file" />

2.  Select a team for the target.

    <img src="/doc_images/Documents/Create_tests/Manage_Projects/Set_Up_Projects/Mobile-Testing/iOS/Perform_Touch_ID_with_iOS_device_simulator/Xcode_-_select_a_team_for_target.png" alt="select a team for the target" />

3.  Press Command + B to build.

4.  Open the folder to get the .app file.

    <img src="/doc_images/Documents/Create_tests/Manage_Projects/Set_Up_Projects/Mobile-Testing/iOS/Perform_Touch_ID_with_iOS_device_simulator/Xcode_-Open_the_folder_to_get_the_.app_file.png" alt="Open the folder to get the .app file" />

5.  You should enroll Touch ID when the simulator is launched by the scripts. Also, it is necessary to define devices that offer Touch ID feature, such as iPhone 8 and iPhone SE.

    <img src="/doc_images/Documents/Create_tests/Manage_Projects/Set_Up_Projects/Mobile-Testing/iOS/Perform_Touch_ID_with_iOS_device_simulator/iOS_-_enroll_Touch_ID.jpeg" alt="enroll Touch ID" />
