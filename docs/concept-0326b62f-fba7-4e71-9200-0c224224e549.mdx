# Katalon Studio Enterprise per-user license

Katalon Studio \(KS\) provides free, basic tools suitable for individuals' testing needs. [Learn more about KS here](/administer/katalon-studio-enterprise-and-katalon-runtime-engine-license/trial-and-free-plans).

Katalon Studio Enterprise \(KSE\) is a more advanced business solution. Purchasing a license allows access to more features. To compare the features between them, see: [Katalon Studio vs Katalon Studio Enterprise Features](/administer/katalon-studio-enterprise-and-katalon-runtime-engine-license/katalon-studio-vs-katalon-studio-enterprise-features).

The table below provides information on the KSE per-user license:
