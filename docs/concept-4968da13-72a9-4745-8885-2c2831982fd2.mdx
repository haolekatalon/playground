# \[Common\] Verify Checkpoint

## Description

Verify if checked data of checkpoint matches their source data.

## Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>checkpoint</td><td>Checkpoint</td><td>Required</td><td>Represent the input checkpoint.</td></tr><tr><td>logChangedValues</td><td>Boolean</td><td>Required</td><td>true will log all the changed values between checkpoint data and the source, false will not log any changed value.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>

## Returns

<table><tr><th>Param Type</th><th>Description</th></tr><tbody><tr><td>Boolean</td><td><ul><li><strong>true</strong>, if checked data of checkpoint matches their source data.</li><li><strong>false</strong>, if checked data of checkpoint NOT matches their source data.</li></ul></td></tr></tbody></table>
