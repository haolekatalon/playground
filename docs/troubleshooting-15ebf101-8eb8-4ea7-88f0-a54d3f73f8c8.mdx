# Invalid element state exception

When you encounter the following exception: <code>org.openqa.selenium.InvalidElementStateException: invalid element state: Element is not currently interactable and may not be manipulated.</code>

1.  Try one of the following solutions to resolve the issue:

    Wait until the element is visible.

    OR

    Set a value directly using JavaScript.

    ```
    import com.kms.katalon.core.webui.common.WebUiCommonHelper 
    WebElement element = WebUiCommonHelper.findWebElement(findTestObject('your/object'),30)
    WebUI.executeJavaScript("arguments[0].value='Your Value'", Arrays.asList(element))
    ```
