# \[Mobile\] Toggle Wifi


:::note

-   Katalon Studio version 8.3.0 onwards.

:::

## Description

Simulate toggling Wifi on/off on mobile devices.


:::note

-   This keyword supports real iOS devices and Android emulators.
-   On iOS devices, **Access Within Apps** must be enabled. To enable the feature, refer to the following Apple document: [Edit Control Center on your iPhone, iPad, and iPod touch](https://support.apple.com/en-us/HT211812).

:::

## Parameters

<table><tr><th>Parameter</th><th>Parameter Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>mode</td><td>String</td><td>Yes</td><td>["no", "off", "false"] to turn off Wifi; ["yes", "on", "true"] to turn on Wifi.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>No</td><td>Specify failure handling schema to determine whether the execution should be allowed to continue or stop. Refer to: <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >Failure Handling</a>.</td></tr></tbody></table>
