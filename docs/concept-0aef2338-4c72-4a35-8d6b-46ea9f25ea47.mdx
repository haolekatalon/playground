# Generate test steps in Katalon Studio script view

Once a new test case is created, you can switch to the **Script view** using the corresponding tab at the bottom of the test case editor. Test steps specified in the **Manual view** are translated into a Groovy script in **Script view**.

<img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Toolbars_and_views/Editor_view/ks-script.png" width={700} alt="Script view" />

By using the import statement in a test script, you can reference to the classes you wish to use. Expand the **import** section to see all default imported classes by Katalon Studio. The name after 'as' in each import statement is an alias for the class. You can change the alias for each class. These classes are necessary for composing a test script.

## Use built-in keywords in script view

Katalon Studio is an automation tool that supports keyword-driven testing. All keywords are grouped into webUI, mobile, web service, windows and cucumber packages accordingly. Press **Ctrl** + **Space** to view these packages and functions from the imported classes.

## Refer to an object in script view

1.  You can find any object ID from its Properties dialog. Go to **Tests Explorer** &gt; **Object Repository**. Right-click on an object and choose **Properties**.

    <img src="/doc_images/Documents/Create_tests/Create_Test_Cases/Images/find-properties.png" width={300} alt="Find object ID" />

2.  To refer to an object in **Object Repository**, use <code>findTestObject('&#10100;Object ID&#10101;')</code>, in which **Object ID** is the ID of that object in Katalon Studio. You can also drag and drop the object into the test case editor to generate this syntax.

    <img src="/doc_images/Documents/Create_tests/Create_Test_Cases/Images/object-properties.png" width={500} alt="Test object properties" />


## Usage example: Generate test steps in script view

In our example, we use a sample test case with the steps as below:

-   Open the browser.
-   Navigate to a website.
-   Click on a specific control.
-   Validate if a control exists on the page.
-   Close the browser.

In this scenario, you will create a Web application test script to make use of the WebUI built-in keywords. Follow these steps to automate the above test scenario in **Script view**:

1.  **Open the browser**: Type <code>WebUI.</code> and select the **[Open Browser](/create-tests/keywords/keyword-description-in-katalon-studio/web-ui-keywords/webui-open-browser)** keyword. This keyword opens a browser and navigates to the specified URL if it is provided. Selected keywords will have their description displayed along for reference.

    <img src="/doc_images/Documents/Create_tests/Create_Test_Cases/Images/open-browser.png" alt="Open browser in script view" />

2.  **Navigate to a website**: Enter the **[Navigate To Url](/create-tests/keywords/keyword-description-in-katalon-studio/web-ui-keywords/webui-navigate-to-url)** keyword. This keyword navigates to a specified URL. For now, enter the URL of Katalon Studio: http://katalon.com/ as the value of the parameter.

    <img src="/doc_images/Documents/Create_tests/Create_Test_Cases/Images/navigate.png" alt="Navigate to a website in script view" />

3.  **Click on a specific control**: Enter the **[Click](/create-tests/keywords/keyword-description-in-katalon-studio/web-ui-keywords/webui-click)** keyword. This keyword represents the click action on a given object. You need to specify an object for this action.

    <img src="/doc_images/Documents/Create_tests/Create_Test_Cases/Images/click.png" alt="Click script view" />

4.  **Validate if a control exists on the page**: Enter the **[Verify Element Present](/create-tests/keywords/keyword-description-in-katalon-studio/web-ui-keywords/webui-verify-element-present)** keyword. This keyword validates if a certain object is displayed on the executing browser. Similar to the previous step, you need to specify the object to be used with this keyword.

    <img src="/doc_images/Documents/Create_tests/Create_Test_Cases/Images/verify-element-present.png" alt="Verify element presents" />

5.  **Close the browser**: Add the **[Close Browser](/create-tests/keywords/keyword-description-in-katalon-studio/web-ui-keywords/webui-close-browser)** keyword and save your test case.

    <img src="/doc_images/Documents/Create_tests/Create_Test_Cases/Images/close-browser.png" alt="Close browser script" />


You have successfully created a test case in Katalon Studio script view.
