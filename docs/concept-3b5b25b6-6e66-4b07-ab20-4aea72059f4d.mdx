# \[WS\] Get Response Status Code

## Description

<code>getResponseStatusCode(ResponseObject responseObject, FailureHandling flowControl)</code>

Get status code in the returned response from a web service call.

## Parameters

<table><tr><th>Parameters</th><th>Parameter Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>response</td><td>ResponseObject</td><td>Required</td><td>The object that represents an HTTP response.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify failure handling schema to determine whether the execution should be allowed to continue or stop. To learn more about failure handling settings, you can refer to this document: <p><a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >Failure handling.</a></p></td></tr></tbody></table>

## Returns

The HTTP response status code.
