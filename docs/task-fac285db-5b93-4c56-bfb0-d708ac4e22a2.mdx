# Search a test case by custom fields and tags

After assigning custom fields and tags to your test cases, you can search test cases in the test case list by filtering custom fields and tags.

1.  Go to **Test Management** &gt; **Test Cases**.

2.  From the **Test Case** page, click the filter icon on the **Search test case** bar.<img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Search_a_Test_Case_by_Custom_Fields_and_Tags/images/TO-Test_Management-test_case_query_filter_icon.png" alt="test case filter" />

3.  Within the filter list, navigate to **Custom Fields** and **Tags**. Select from the dropdown list below each of them the custom fields and tags based on which you want to query your test case. Click **Apply and search** to search for test cases with selected custom fields and tags.

    <img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Search_a_Test_Case_by_Custom_Fields_and_Tags/images/TO-Test_Management-custom_field_and_tags_filter.png" alt="search test case by custom fields and tags" />


Your search result displays all test cases with selected custom fields and tags.<img src="/doc_images/Documents/Organize/Manage_Tests/Test_case/Search_a_Test_Case_by_Custom_Fields_and_Tags/images/TO-Test_Management-test_case_search_result.png" alt="test case search result" />
