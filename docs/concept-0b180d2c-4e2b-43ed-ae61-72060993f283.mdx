# \[WebUI\] Set Alert Text

## Description

Simulate users typing text into a prompt popup.

## Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>text</td><td>String</td><td>Required</td><td>The text to type</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>

You want to input "Sample Alert Text" text into a prompt popup
