# \[WS\] Verify Response Status Code

## Description

Verify status code in the returned data from a web service call.

Keyword name: <code>WS.verifyResponseStatusCode</code>

## Parameters

<table><tr><th>Parameter</th><th>Parameter Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>responseObject</td><td>ResponseObject</td><td>Required</td><td>The object that represents an HTTP response.</td></tr><tr><td>expectedStatusCode</td><td>int</td><td>Required</td><td>The expected status code</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>

## Returns

-   <code>true</code> if the response status code is the same as the expected status code, otherwise <code>false</code>.
