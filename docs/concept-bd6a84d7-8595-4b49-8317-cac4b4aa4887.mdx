# \[Common\] Verify Match

## Description

Verify if two strings match each other, the second string can be a regular expression.

## Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>actualText</td><td>String</td><td>Required</td><td>Represent the actual text.</td></tr><tr><td>expectedText</td><td>String</td><td>Required</td><td>Represent the expected text (can be a regular expression).</td></tr><tr><td>isRegex</td><td>Boolean</td><td>Required</td><td>Indicate whether the expected text is a regular expression.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>

## Returns

<table><tr><th>Param Type</th><th>Description</th></tr><tbody><tr><td>Boolean</td><td><strong>true</strong>, if two given strings match each other; otherwise, <strong>false</strong>.</td></tr></tbody></table>
