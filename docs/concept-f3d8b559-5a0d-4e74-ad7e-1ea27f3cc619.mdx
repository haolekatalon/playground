# \[Common\] Verify Less Than Or Equal

## Description

Verify if the actual number is less than or equal to the expected number.

## Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>actualObject</td><td>Object</td><td>Required</td><td>Represent the actual object.</td></tr><tr><td>expectedObject</td><td>Object</td><td>Required</td><td>Represent the expected object.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>

## Returns

<table><tr><th>Param Type</th><th>Description</th></tr><tbody><tr><td>Boolean</td><td><ul><li><p><strong>true</strong> if the actual number is less than or equal to the expected number.</p></li><li><p><strong>false</strong> if the actual number is NOT less than or equal to the expected number.</p></li></ul></td></tr></tbody></table>
