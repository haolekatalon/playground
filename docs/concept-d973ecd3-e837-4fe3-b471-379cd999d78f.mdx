# \[Mobile\] Perform Touch ID


:::note

-   Katalon Studio version 8.3.0 onwards.

:::

## Description

Simulate a Touch ID event on iOS simulators.


:::note

-   To enable this keyword, Touch ID on the simulator must be enrolled.
-   Touch ID and Face ID are available on certain iPhone and iPad models. To learn more about supported iPhone and iPad models, refer to the following Apple documents: [iPhone models](https://support.apple.com/en-us/HT201296) and [iPad models](https://support.apple.com/en-us/HT201471).

:::

## Exceptions

<table><tr><th>Throw</th><th>Precondition</th><th>Message</th></tr><tbody><tr><td><code>StepFailedException</code></td><td>If Touch ID on the simulator is not enrolled.</td><td>To enable this keyword, touch ID on the simulator must be enrolled.</td></tr><tr><td><code>StepFailedException</code></td><td>If the <code>performTouchID</code> keyword is not performed on an iOS simulator.</td><td>The <code>performTouchID</code> keyword supports iOS simulators only.</td></tr><tr><td><code>Warning</code></td><td>If the <code>performTouchID</code> keyword is run on unsupported environments.</td><td>The <code>performTouchID</code> keyword is not available for physical mobile devices.</td></tr></tbody></table>
