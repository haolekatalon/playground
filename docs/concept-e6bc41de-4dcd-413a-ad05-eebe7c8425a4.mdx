# Sample projects overview


:::note

-   Katalon Studio installed and activated. To learn more about installing and activating Katalon Studio, you can refer to this document: [Installation](/get-started/katalon-studio-installation/katalon-studio-installation-overview).

:::

To help you get familiar with Katalon Studio capabilities, Katalon has created sample projects for each supported testing types. You can open those sample projects in Katalon Studio.
