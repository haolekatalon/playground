# Katalon Studio - Common steps

-   From the main toolbar, click **Create new Test Case**.

    Or, in the **Tests Explorer** panel, right-click the **Test Case** folder then select **New** &gt; **Test Case**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.3.5-Create-new-tc.png" width={500} alt="Create new test case" />

    Alternatively, you can also go to **File** &gt; **New** &gt; **Test Case** from the main menu.

    
    :::tip

    -   For better management, you can also create a new folder by selecting **Test Cases** &gt; **New** &gt; **Folder**, then create new test cases inside that folder.

    :::

-   Go to **File** &gt; **New** &gt; **Project** from the main menu.

-   In the **Tests Explorer** panel, right-click on the **Object Repository** folder, then select **New** &gt; **Test Object**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-a-test-object.png" width={500} alt="Create a new test object" />

    Alternatively, you can also go to **File** &gt; **New** &gt; **Test Object** from the main menu.

    
    :::tip

    -   For better management, you can also create a new folder by selecting **Test Suites** &gt; **New** &gt; **Folder**, then create new test suites inside that folder.

    :::

-   Select **File** &gt; **Open Project** from the main menu.

-   In the **Tests Explorer** panel, right-click on the **Test Suites** folder &gt; **New** &gt; **Test Suite**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-test-suite.png" width={500} alt="Create a new test suite" />

    Alternatively, you can also go to **File** &gt; **New** &gt; **Test Suite** from the main menu.

-   In the **Tests Explorer** panel, right-click on the **Test Suites** folder &gt; **New** &gt; **Test Suite Collection**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-tsc.png" width={500} alt="Create a new test suite collection" />

    Alternatively, you can also go to **File** &gt; **New** &gt; **Test Suite Collection** from the main menu.

    A **New Test Suite Collection** dialog appears.

-   In the **Tests Explorer** panel, right-click at the **Test Suites** folder &gt; **New** &gt; **Dynamic Test Suite**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-dynamic-ts.png" width={500} alt="Create a new dynamic test suite" />

-   In the **Tests Explorer** panel, right-click on the **Object Repository** folder, select **New** &gt; **Web Service Request**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-web-service-request.png" width={500} alt="create a web service request" />

    Alternatively, you can also go to **File** &gt; **New** &gt; **Web Service Request** from the main menu.

-   In the **Tests Explorer** panel, right-click on the **Data Files** folder &gt; **New** &gt; **Test Data**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-test-data.png" width={400} alt="create a new test data" />

    Alternatively, you can also go to **File** &gt; **New** &gt; **Test Data** from the main menu.

-   In the **Tests Explorer** panel, right-click on the **Keywords** folder &gt; **New** &gt; **Package**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-package.png" width={500} alt="Create a package" />

    Alternatively, you can also go to **File** &gt; **New** &gt; **Package** from the main menu.

-   In the **Tests Explorer** panel, right-click on the **Keywords** folder &gt; **New** &gt; **Keyword**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-keyword.png" width={500} alt="Create a new keyword" />

-   Go to **File** &gt; **New** &gt; **Groovy Script** from the main menu.

-   Go to **File** &gt; **New** &gt; **BDD Feature File** from the main menu.

-   In the **Tests Explorer** panel, right-click on the **Checkpoints** &gt; **New** &gt; **Checkpoint**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-checkpoint.png" width={500} alt="Create a new checkpoint" />

    Alternatively, you can go to **File** &gt; **New** &gt; **Checkpoint** from the main menu.

-   In the **Tests Explorer** panel, right-click on **Profiles**. Select **New** &gt; **Execution Profile**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-execution-profile.png" width={500} alt="Create a new execution profile" />

-   In the **Tests Explorer** panel, right-click on **Test Listeners**. Select **New** &gt; **New Test Listener**.

    <img src="/doc_images/Global_reuse/Images_warehouse/Katalon_Studio/Tests_Explorer/KS-8.2.5-create-test-listener.png" width={500} alt="Create a new test listener" />

-   Return to Katalon Studio and activate your plugin. To do so, click on the *Profile* icon, then click **Reload Plugin**. If you want to use the plugin in console mode, refer to this document: [Use Plugins in Console Mode](https://docs.katalon.com/katalon-studio/docs/kse-use-plugins.html#use-plugins-in-console-mode).

-   Open a test suite. Click **Add**. A **Test Case Browser** opens. Choose one or more test cases you wish to add, then click **OK**.

-   Open a test suite. Click **Add**. A **Test Case Browser** opens.

-   Open a test suite collection. Click **Add**. A **Test Suite Browser** opens. Choose one or more test suites you wish to add, then click **OK**.

-   Open a test suite collection. Click **Add**. A**Test Suite Browser** opens.

-   To execute a test suite, from the main toolbar, click **Run**.

-   To run a test suite collection, from the main toolbar, click **Execute**.

-   To execute a dynamic test suite, from the main toolbar, click **Run**.

-   To execute a test case, from the main toolbar, click **Run**.

    <img src="/doc_images/Global_reuse/Common_steps/image/ks-run.png" width={250} />

-   From the main toolbar, click on **Record Web** to open the **Web Recorder**.

    <img src="/doc_images/Global_reuse/Common_steps/image/ks-record-web.png" width={400} />

    The **Web Recorder** dialog appears.<img src="/doc_images/Global_reuse/Common_steps/image/ks-web-recorder-dialog.png" width={700} />

-   Go to **Project Settings** &gt; **Library Management**. Click **Add** button to add the jar file to the external library.

-   On your macOS computer, locate to the folder containing Katalon Studio. Then right click at the Katalon Studio icon, select **Show Package Contents**.

    <img src="/doc_images/Global_reuse/Common_steps/image/Katalon_Studio_packet_content.png" alt="Katalon Studio package contents" />

    Then open the folder **Contents/ Eclipse**, there you will find the katalon.ini file, open it in a text editor.

    On your Windows OS computer, open the folder containing Katalon Studio, there you will find the katalon.ini file, open it in a text editor.

-   In **Project Settings**, go to **Database** to configure the database connection.

    1.  Select **Secure User and Password** to enable **User** and **Password** fields.

    2.  Input the **User** and **Password** used for authentication.

    3.  Enter **Connection URL**.

    4.  Click **Test Connection** to verify whether your database is connected successfully.
