# \[WebUI\] Mouse Over Offset

## Description

Simulate users hovering a mouse over the given element with the relative position \(x, y\) from the top-left corner of that element.

Parameters

<table><tr><th>Param</th><th>Param Type</th><th>Mandatory</th><th>Description</th></tr><tbody><tr><td>to</td><td>TestObject</td><td>Required</td><td>Represent a web element.</td></tr><tr><td>offsetX</td><td>int</td><td>Required</td><td>x position in relative to the element.</td></tr><tr><td>offsetY</td><td>int</td><td>Required</td><td>y position in relative to the element.</td></tr><tr><td>flowControl</td><td>FailureHandling</td><td>Optional</td><td>Specify <a href="/maintain/configure-failure-handling-settings-in-katalon-studio" >failure handling</a> schema to determine whether the execution should be allowed to continue or stop.</td></tr></tbody></table>
