# Configure Xray Integration in Katalon TestOps

Xray is a complete Test Management tool for Jira. It is an Atlassian marketplace application and runs natively as part of Jira.

With Xray integration you will be able to:

-   Link Xray test case with Katalon test case

-   Link Xray test plan with Katalon test run

-   Manually or automatically upload Katalon test result to Xray

-   Link Jira fix version and test evidence to uploaded Katalon test result

This article shows you how to configure Xray integration in Katalon TestOps.

## Prerequisites

-   Your Jira project must have an active Xray configuration. See [Xray Test Management for Jira on Atlassian Marketplace](https://marketplace.atlassian.com/apps/1211769/xray-test-management-for-jira).

-   You must have the Admin role in your Katalon project. See [Administrative Roles and Permissions](/administer/administration-roles/administrative-roles-and-permissions).


## Configure Xray integration in Katalon TestOps

1.  Sign in to [Katalon TestOps](https://testops.katalon.io/login) and go to your project.

    The project **Dashboard** page appears.

2.  Select **Configuration** &gt; **Integration**.

    The **Integrations** page appears. There is a dropdown menu on the page for a full list of integration options.

3.  Select **Jira/Xray** in the dropdown menu.

    The Jira page appears as below.<img src="/doc_images/Documents/Organize/Integration_For_Organizing_Tests/Xray_Integration/Configure_Xray_integration_in_TestOps/image/TestOps_integrations_page.png" alt="TestOps integration page" />

4.  Enter your Jira URL, Username, and Password into the integration page:

    -   Jira **URL** must be in the form of <code>https://&lt;site&#95;name&gt;.atlassian.net</code>.

    -   In the **Username** section, enter the email address you have registered for your Jira account.

    -   In the **Password** section, enter an Atlassian Cloud's API token. To retrieve your Atlassian API tokens, see: [API tokens](https://confluence.atlassian.com/cloud/api-tokens-938839638.html).

5.  Toggle on **Enable Xray integration**.

    <img src="/doc_images/Documents/Organize/Integration_For_Organizing_Tests/Xray_Integration/Configure_Xray_integration_in_TestOps/image/TO-Integration-enable_Xray_toggle.png" alt="toggle on Xray integration" />

6.  In the **Client ID** section, fill in with your Xray API key. To retrieve your Xray API, see [Xray API key](https://docs.getxray.app/display/XRAYCLOUD/Global+Settings%3A+API+Keys).

7.  Click on **Test Connection** and make sure the Jira/Xray and Katalon connection is successful.

    <img src="/doc_images/Documents/Organize/Integration_For_Organizing_Tests/Xray_Integration/Configure_Xray_integration_in_TestOps/image/TO-Integration-connect_successfully_dialog.png" />

8.  In the **Link Project** section, select from the dropdown list your Jira project.

    <img src="/doc_images/Documents/Organize/Integration_For_Organizing_Tests/Xray_Integration/Configure_Xray_integration_in_TestOps/image/TO-Integration-Xray_integration-Link_Project.png" />

9.  Select **Save**.


You have successfully enabled TestOps - Jira/Xray integration in Katalon TestOps.
