# Set up overview in Katalon Platform

This article guide you through setting up your workspace to start working with Katalon Platform.

## Integrate Katalon TestOps with Katalon Studio

In Katalon Platform, you journey starts with Katalon TestOps. This document shows you how to get started with Katalon TestOps: [Create Account, Organization, and Projects](/administer/administration-tasks/create-an-organization-and-project).

If you are using Katalon Studio, refer to this document to see how to integrate Katalon Studio with Katalon TestOps for test planning, management, reporting and analytics: [Integrate Katalon TestOps with Katalon Studio](/get-started/set-up-your-workspace/integrate-katalon-testops-and-katalon-testcloud-with-katalon-studio).

If you are using Katalon Studio - Platform Edition, Katalon TestOps is integrated by default. You can open your project and start testing.
